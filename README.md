# Jogo "Quem é este pokémon"?
Esse é um pequeno jogo que utiliza a API [Poké API](https://pokeapi.co) onde o jogador deve adivinhar o pokémon de acordo com a sua silhueta. O projeto é de nível fácil, mas é um bom treino para quem quiser aprender a utilizar a função **fetch** e utilizar funções assíncronas com async e await.

## Pontos Treinados
* Manipulação de DOM
* Uso de Fetch, Await e Async
* Estruturação de HTML e estilização básica com CSS
